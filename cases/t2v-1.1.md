# T2V 文生视频案例（HappyHorse 1.1）

> HappyHorse 1.1 T2V 支持更长更复杂的文本输入，2500 字符后的指令遵循度显著提升。单条 Prompt 可描述 6-8 个连续场景，模型自动分配时间、切换镜头；双人/多人场景中的空间位置关系和交互逻辑也更加准确。

**适用场景：**
- 复杂叙事短片、多场景连续调度
- 双人/多人互动、对白驱动的情绪戏
- 广告片、动漫、游戏 CG 等风格化生成

## T2V 1.1 创作提示

- 使用 `[00:00-00:04]` 时间码进行多场景精确调度
- 复杂场景建议明确角色位置关系和镜头运动
- 1.1 对长上下文理解更强，可适当增加细节和环境描述
- 音画同步：可在 Prompt 末尾追加音效/台词/背景音乐描述
