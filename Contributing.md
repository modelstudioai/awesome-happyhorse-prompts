# Contributing

欢迎提交高质量的 HappyHorse 1.0 Prompt 案例、API 使用技巧、创意工作流或文档改进。

## 提交指南

- 保持案例精选，避免重复。
- 优先提交可复用的 Prompt 模板或 API 调用示例。
- 更新 README 时保持结构一致。
- 视频案例请使用 GitHub 附件链接（`https://github.com/user-attachments/assets/...`），确保文件小于 10MB。

## 案例格式

每个案例请包含：

1. **标题**：简短描述场景
2. **Prompt**：完整的提示词（中文或英文）
3. **参数**：模型、分辨率、时长等
4. **输出效果**：视频或图片链接

示例：

```markdown
### Case 1: 古风玄幻场景

**Prompt:**
```
一位身穿白色长袍的仙子，站在云雾缭绕的山峰之上，衣袂飘动，远处仙鹤飞过...
```

**参数：** T2V-14B | 1080P | 10s

https://github.com/user-attachments/assets/xxxx-xxxx-xxxx
```

## 提交方式

1. Fork 本仓库
2. 在对应分类的 `cases/*.md` 文件中添加案例
3. 提交 Pull Request

感谢你的贡献！
